###### NAME_OF_LISTED

n = REQUEST("NAME_OF_LISTED", "", name) Returns in the name variable the name of the library part associated with the property type library part containing this request. For elements (Walls, Slabs, etc.), the name is an empty string. Causes warning if used in parameter script.
