###### NAME_OF_PROFILE

n = REQUEST("NAME_OF_PROFILE", index, name) Returns in the name variable the profile name identified by index. Expression returns 0 and contains dummy return values (empty string or 0) if used in parameter script, causing additional warning.

- Compatibility: introduced in Archicad 21.
