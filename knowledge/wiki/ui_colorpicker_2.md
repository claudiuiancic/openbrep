###### UI_COLORPICKER{2}

UI_COLORPICKER{2} redParamName, greenParamName, blueParamName, x0, y0 [, width [, height]] Color picker dialog to set the r, g, b components of a color and store them into the given parameters. These values can later be used in the LIGHT command. redParamName, greenParamName, blueParamName: parameter names as string expression for UI_COLORPICKER or

parameter names with optional actual index values if array for UI_COLORPICKER{2} x0, y0: position of the color picker's top left corner.

width, height: width and height in pixels.
