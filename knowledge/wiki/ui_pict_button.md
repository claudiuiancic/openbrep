###### UI_PICT_BUTTON

UI_PICT_BUTTON type, text, picture_reference,

x, y, width, height [, id [, url]] Similar to the UI_BUTTON command. But this type of buttons can contain pictures. picture_reference: file name or index number of the picture stored in the library part. The index 0 refers to the preview picture of

the library part. Pixel transparency is allowed in the picture. text: has no effect for picture buttons.
