#### FILE OPERATIONS

The following keywords allow you to open external files for reading/writing and to manipulate them by putting/getting values from/to GDL scripts. This process necessarily involves using special Add-On extensions. Text files can be handled by the section called “GDL Text I/O AddOn”. Add-Ons for other file types can be developed by third parties.

See also the section called “GDL Text I/O Add-On”.
