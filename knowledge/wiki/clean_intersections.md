###### CLEAN_INTERSECTIONS

n = REQUEST("CLEAN_INTERSECTIONS", "", state) Returns the state of the Clean Wall & Beam Intersections feature (1 when turned on, 0 when off) Expression returns 0 and contains dummy return values (empty string or 0) if used in parameter script, causing additional warning.
