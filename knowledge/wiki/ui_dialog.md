###### UI_DIALOG

UI_DIALOG title [, size_x, size_y] Defines the title of the dialog box. The default title is 'Custom Settings'. Currently, the size of the available area is fixed at 444 x 296 pixels, and the size_x and size_y parameters are not used. Restriction: The Interface Script should contain only one UI_DIALOG command.
