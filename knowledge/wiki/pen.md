###### PEN

PEN n Sets the color. Restriction of parameters:

0 < n <= 255 Default: PEN 1 if there is no PEN statement in the script. (For library parts, default values come from the library part’s settings. If the script refers to a non-existing index, PEN 1 becomes the default setting.)
