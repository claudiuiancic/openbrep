###### VIEW_ROTANGLE

n = REQUEST("VIEW_ROTANGLE", "", angleViewRotation) Returns the rotation angle of the current view. Expression returns 0 and contains dummy return values (empty string or 0) if used in parameter script, causing additional warning.
