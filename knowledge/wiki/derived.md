#### DERIVED TYPES

Variables and parameters can also be arrays, and parameters can be value lists of a simple type. Arrays are one- or two-dimensional tables of numeric and/or string values, which can be accessed directly by indexes. Value lists are sets of possible numeric or string values. They can be assigned to the parameters in the value list script of the library part or in the MASTER_GDL script, and will appear in the parameter list as a pop-up menu.
