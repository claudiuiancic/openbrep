###### PARVALUE_DESCRIPTION

PARVALUE_DESCRIPTION (parname [, ind1 [, ind2]]) This function returns the parameter value description string of a numerical parameter specified using the VALUES command statement. If no description is specified, the returned value is an empty string.

parname: name of the parameter ind1, ind2: actual indices if the parameter is an array.
